# The Hive

The Hive is a community-oriented service exchange platform where people help each other by offering and requesting services.  
Instead of money, members exchange **TimeBank hours** — 1 hour of service = 1 hour earned.

## Features
- Post Offers and Needs
- Semantic search and recommendation
- Real-time messaging
- Handshake-based service confirmation
- TimeBank hour accounting (no currency)
- Mutual rating system
- Community forum
- Admin moderation tools
